# タイピングリズム - TYPING RHYTHM

**A retro pixel-style typing rhythm game for Windows**

## 🎮 概要 (Overview)

A Japanese-style rhythm game where you type falling letters to the beat! 
Inspired by classic arcade rhythm games with a typing twist.

## ✨ Features

- **Pixel Art Style** - Retro 8-bit visuals with pixel-perfect rendering
- **Chiptune Soundtrack** - Original Japanese-style chiptune BGM with Web Audio API
- **Typing Gameplay** - Type the correct letter when it hits the judgment zone
- **Combo System** - Chain hits for bonus points!
- **PERFECT / GOOD / MISS** - Precision-based scoring
- **Japanese Game Aesthetic** - Traditional arcade feel with Japanese text

## 🎯 遊び方 (How to Play)

1. Open `typing_rhythm.html` in your browser (Chrome/Edge recommended)
2. Press **ENTER** to start (or click the screen)
3. Type the letters **on the falling notes** when they reach the red line!
4. Try to get **PERFECT** for maximum points
5. Keep the combo going for huge scores!

## ⌨ Key Bindings

- **A, S, D, F, G, H, J, K, L** - Main typing keys
- **Q, W, E, R, T, Z, U, I, O, P, Y, X, C, V, B, N, M** - Advanced keys
- **ENTER** - Start / Restart
- **; (semicolon)** - Also a valid key

## 🎵 Score System

| Judge | Points | Effect |
|-------|--------|--------|
| PERFECT (パーフェクト!!) | 150 + combo×8 | Gold flash, special sound |
| GOOD | 70 + combo×4 | Green flash, hit sound |
| MISS | 10 | Combo break |

## 🖥 Requirements

- Windows (or any OS with a modern browser)
- Keyboard
- Web Audio API compatible browser
